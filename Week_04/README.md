## 字符串分析算法
> ll 和 正则前面都用过了，这节课主要学习 字典树，KMP，Willcard
- 字典树
    - 检查两个字符串是否完全匹配
    - 适用于大量高重复字符串的存储和分析
- KMP
    - 检查两个字符串是否部分匹配
    - 字符串中查找短字符串（模式）
- Wildcard
    - 包含通配符的字符串模式匹配
- 正则表达式
    - 字符串通用模式匹配
- 状态机
    - 通用的字符串分析
- LL/LR
    - 字符串多层次结构分析
